# AeroLab 航空模拟教学平台 Demo

一个面向中小学科普场景的轻量航空模拟平台。Demo 聚焦基础飞行力学：用户可以调节油门、迎角、襟翼、起落架和风速，观察飞机从滑跑到起飞、爬升、巡航、进近的状态变化。

## 功能

- 二维航空模拟场景：跑道、天空、飞机姿态和四种力矢量。
- 实时数据面板：升力、重力、推力、阻力、速度、高度、垂直速度。
- 教学引导：按阶段提示“为什么飞机能飞”。
- 风切变与失速风险：展示错误操作的即时反馈。
- Java 后端：提供教学步骤、默认场景和单步模拟计算接口。
- Vue 前端：白色毛玻璃界面、动态过渡、可交互控制台。

## 本地启动

先启动 Java 后端：

```powershell
.\run-backend.cmd
```

再启动 Vue 前端：

```powershell
.\run-frontend.cmd
```

前端默认地址：`http://127.0.0.1:5173`

后端默认地址：`http://127.0.0.1:8088`

## 目录

```text
backend/   Java 原生 HTTP 服务
frontend/  Vue + Vite 前端
```
